# WMS Notes

WMS Notes is een heel handig programma om notities en aantekeningen te bewaren. Je kan heel snel een nieuwe notitie maken door op één van de knoppen in de knoppenbalk te klikken en je kan een onbeperkt aantal notities maken! Om alles overzichtelijk te houden worden de notities weergegeven in een handige boomstructuur waarin je groepen maken om de notities in te plaatsen. Ook kan je door deze boomstructur makkelijk overschakelen tussen de verschillende notitites.

Je kan gebruik maken van alle bekende opmaakmethoden om de notities er zo goed mogelijk uit te laten zien. Door middel van een snelle zoekfunctie is het mogelijk om elke notitie altijd heel snel terug te vinden.

Er kunnen meerdere notesbestanden gebruikt worden, die elk met een wachtwoord versleuteld kunnen worden.

WMS Notes gebruikt HTML Help en een aantal extra bestanden (zie VBRun60Complete.exe).
